Personal Profile Website

Project Description
This is a responsive personal profile website built using HTML, CSS, and Bootstrap.  
It includes a navigation bar, sections (About, Skills, Education, Contact), buttons, and a working modal.

Responsive navigation bar
 About section with modal popup
 Skills section
 Education section
 Contact section
 Bootstrap buttons
 External CSS styling
